🔑 GitHub 열쇠(토큰) 설정

웹사이트가 GitHub에 저장하려면 "열쇠"가 한 번 필요해요.
1. 이 링크를 새 탭에서 열기 (GitHub 로그인 필요)
2. Token name: yeonmyu / Expiration: No expiration
3. Repository access: Only select repositories → yeonmyu-quotes 선택
4. Permissions → Repository permissions → Contents를 Read and write로
5. 맨 아래 Generate token → 나온 github_pat_... 글자를 복사해 아래에 붙여넣기

이 열쇠는 지금 이 기기의 브라우저에만 저장돼요. 아이폰에서도 쓰려면 아이폰에서 한 번 더 붙여넣으면 돼요.
